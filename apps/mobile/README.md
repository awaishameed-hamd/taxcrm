# CA Firm CRM — Mobile App (Phase 2)

React Native + Expo

> **Status:** Not started. Will be built after the web CRM is complete and stable.

## Planned Stack
- React Native + Expo SDK 51
- Expo Router (file-based routing)
- NativeWind (Tailwind for RN)
- Socket.IO client (same real-time chat as web)
- Expo Notifications (push)

## Roles
All roles supported (Partner, Manager, Trainee, Client)
Primary device for **Clients**, secondary for internal staff.
